# Fibonacci API

โปรเจคนี้เป็นแอปพลิเคชัน Node.js ง่ายๆ ที่ให้ API สำหรับคำนวณลำดับฟีโบนัชชีจนถึงจำนวนสมาชิกที่กำหนด

## เริ่มต้นใช้งาน
1.ทำการติดตั้ง
2.ทำการโหลด
### ข้อกำหนดเบื้องต้น

- [Node.js](https://nodejs.org/) (เวอร์ชัน 14 หรือสูงกว่า)
- [npm](https://www.npmjs.com/)
